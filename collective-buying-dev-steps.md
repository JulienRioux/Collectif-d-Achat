# 🛠️ Development Steps — Collective Buying MVP

## Phase 1 — Setup

1. Create Supabase project  
2. Create tables (organizations, products, baskets, orders)  
3. Add indexes and constraints  
4. Seed initial products  

---

## Phase 2 — Admin Core

1. Build admin auth (Supabase Auth)  
2. Products CRUD  
3. Weekly basket CRUD  
4. Add basket items  
5. Implement basket duplication  

---

## Phase 3 — Ordering Flow

1. Public order page  
2. Form validation (React Hook Form + Zod)  
3. Server endpoint for order submission  
4. Recalculate totals server-side  
5. Enforce budget limit  

---

## Phase 4 — Aggregation

1. Orders list page  
2. Totals by product query  
3. Distribution list view  
4. Revenue / progress view  

---

## Phase 5 — Operations

1. CSV export (orders + totals)  
2. Print-friendly summary page  
3. Basket open/close toggle  

---

## Phase 6 — Polish

1. UI improvements (shadcn/ui)  
2. Sorting and filtering  
3. Error handling  
4. Basic logging  

---

## Phase 7 — Future (Optional)

- Multi-org UI onboarding  
- Email notifications  
- Forecasting  
- Supplier integrations
